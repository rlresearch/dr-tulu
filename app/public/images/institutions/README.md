# Institution Logos

This directory should contain the logos for the following institutions:

## Required Logo Files

1. **uw-logo.png** - University of Washington
   - Download from: https://www.washington.edu/brand/graphic-elements/logo-download/
   - Use the primary horizontal logo in PNG format

2. **ai2-logo.png** - Allen Institute for AI
   - Download from: https://allenai.org/
   - Look for brand assets or press kit section
   - Recent rebrand info: https://allenai.org/blog/rebrand

3. **mit-logo.png** - Massachusetts Institute of Technology
   - Download from: https://web.mit.edu/graphicidentity/
   - Use the official MIT logo

4. **cmu-logo.png** - Carnegie Mellon University
   - Download from: https://www.cmu.edu/brand/brand-guidelines/visual-identity/logos.html
   - Use the primary horizontal logo

5. **ucb-logo.png** - UC Berkeley
   - Download from: https://brand.berkeley.edu/logos/
   - Use the UC Berkeley primary logo

6. **sch-logo.png** - Seattle Children's Hospital
   - Download from: https://www.seattlechildrens.org/
   - Contact their media/press department for official logos

## Logo Specifications

- **Format**: PNG with transparent background preferred
- **Size**: Approximately 240px width, maintain aspect ratio
- **Color**: Use full-color versions where available

## How to Add Logos

1. Download each logo from the official sources above
2. Rename them according to the filenames listed
3. Place them in this directory (`/public/images/institutions/`)
4. Uncomment the Image component code in `/src/app/page.tsx` (lines 156-167)

## Alternative: Using SVG

If you prefer SVG format for better scalability:
1. Download SVG versions where available
2. Update the file extensions in the component
3. Adjust the Image component configuration accordingly
